This plugin allows you to display cookie notice, enter notice text and style it.

== Description ==
This plugin was created with simplicity in mind, it\'s main goal is to display cookie notices without much hassle. All you need to use it, is activating it. Later you can customize settings to your desire.
In case using WPML (The WordPress Multilingual Plugin), you can have different settings depending on language you choose.
The plugin is translation ready.

== Installation ==
1. Download plugin
2. Activate plugin
3. Use \'Cookie notice\' submenu in options to customize(optional)
